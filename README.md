## Installation

To get started with the project, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/HarshittSinghh/appyfood.git
    ```
2. Navigate to the project directory:
    ```bash
    cd appyfood
    ```
3. Install dependencies:
    ```bash
    flutter pub get
    ```
4. Run the app:
    ```bash
    flutter run
    ```
